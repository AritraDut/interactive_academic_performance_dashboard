# Interactive Academic Performance Analytics Dashboard

## Project Overview
This project is an interactive academic performance analytics dashboard developed using Python, SQLite/MySQL, and Power BI. The system helps educational institutions analyze student performance, attendance, CGPA trends, and identify at-risk students through data visualization and analytics.

---

## Technologies Used
- Python
- Pandas
- NumPy
- SQLite / MySQL
- Power BI
- Matplotlib

---

## Features
- Student Performance Analysis
- Attendance Monitoring
- GPA & CGPA Tracking
- Risk Prediction
- Department-wise Analysis
- Interactive Dashboard Visualizations
